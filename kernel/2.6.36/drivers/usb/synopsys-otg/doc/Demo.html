Running Demo Application on NC Platform (IPMate)


Run the provided "demo" script (located in the /root/bin/ directory of root-ipmate.cramfs) to start the demo and load appropriate driver modules.

During the demo, there are two file list boxes. The left one represents the host machine. The files on the /mnt/localfs are shown here. The right one shows the files on an attached USB mass storage device. You can open image files from either side, and also transfer files back and forth. Once you've opened an image, you can print or close the image.

Printing is enabled by attaching an HP photosmart 240 printer on the bus. Printing directly from a file on a USB device is only supported if a hub is
attached.

It may take some amount of time for the demo to detect attachment and removal of USB devices. Do not remove the USB device while transferring files, printing, or
picture loading.


Detailed Demo Procedure

Equipment

- IPMate platform

- Sony Cyber-shot DSC-P41 digital camera

- HP Photosmart 245 printer

- Windows XP host

- Cable with mini-A connector on one end and mini-B connector on the other for the camera portion of the host demo.

- Cable with mini-A connector on one end and a male-B connector on the other for doing the printer portion of the host demo.

- Cable with a mini-B connector on one end and a male-A connector on the other for doing the device demo.

Setup

- Take some pictures with the digital camera.

- Connect a mouse to the IPMate. (This is optional, but a mouse may be useful if the touch screen is not aligned properly.)


Demo Procedure

- Start with mini-A connector and cable connected to the IPMate and a mini-B connector connected to the camera.

- Power on the IPMate. Use a serial console to control the demo.

- Log in as root.

- Execute "demo" or "demo mouse". The former uses the touch screen on the IPMate. The latter allows using a mouse.

- Wait for the GUI to appear on the LCD. The drivers load after the GUI and they take about 5 seconds to load.

- If the camera was turned on at the start of the demo, the application should detect the device after the drivers load and display the top level directory from the camera. Otherwise, turn on the camera now and the demo will detect it and display the top level directory.

- Navigate to the dcim/101msdcf directory (or something similar) by using the Open button on the device side of the screen.

- Transfer some files from the camera to the Local Files area.

- Choose one of the pictures in Local Files and Open it.

- When the picture viewer opens, disconnect the mini-A to mini-B cable and connect the mini-A to male-B cable between the IPMate and the
printer.

- Make sure the printer is on. Press the Print button on the application to print the picture. You can close the picture after printing.

- Remove the mini-A cable from the IPMate. Attach the mini-B to male-A connector between the IPMate and the Windows XP host. The IPMate should appear as a mass storage device in Windows Explorer.

- View pictures transferred from the camera on the Windows XP host.

- Demo complete!
